# Romantic Birthday Apology App

A mobile-first Next.js App Router birthday and apology website built with TypeScript, Tailwind CSS, and Framer Motion.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL printed by Next.js, usually `http://localhost:3000`.

## Media

The app uses the files currently stored in `public`:

- `/1.jpeg`
- `/2.jpeg`
- `/3.jpeg`
- `/4.jpeg`
- `/apology.mp4`

## Editable Content

Update the quiz answers, secret code, and romantic letter in:

```text
components/content.ts
```

The secret code is `909134`.
